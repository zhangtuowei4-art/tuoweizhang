# TMI 2024 Knowledge Graph Visualization

## 项目简介
这是一个知识图谱可视化网页，专为展示 **TMI 2024 (IEEE Transactions on Medical Imaging)** 期刊论文的关联关系而设计。

## 核心功能
1.  **交互式图谱**：使用 ECharts 力导向图展示大规模节点与连线，支持缩放、平移和拖拽。
2.  **多维度搜索**：支持通过关键词（论文标题、作者、方法等）实时过滤，高亮显示匹配的节点及其关联关系。
3.  **详情展示与联动**：
    *   **点击论文节点**：在侧边栏显示详细信息（DOI、创新点、指标等），并在图谱中高亮其直接关联的邻居节点。
    *   **点击实体节点**（如 "Segmentation"）：侧边栏列表自动筛选出所有包含该实体的相关论文。
4.  **性能优化**：针对力导向图的高亮交互进行了深度优化，避免了因节点大小变化引起的布局重算卡顿。

## 技术栈
*   **HTML5**: 页面语义化结构。
*   **CSS3**: 响应式布局，暗色侧边栏 UI 设计。
*   **JavaScript (ES6+)**: 数据处理逻辑、DOM 操作及事件绑定。
*   **ECharts 5.4.3**: 用于渲染高性能交互式图谱。

## 文件说明

| 文件名 | 说明 |
| :--- | :--- |
| `index.html` | 页面主结构。包含左侧侧边栏（搜索框、详情面板、论文列表）和右侧图表主容器。 |
| `style.css` | 全局样式表。定义了应用的主题色、侧边栏布局、加载动画及卡片样式。 |
| `charts.js` | ECharts 初始化配置文件。定义了图谱的力导向参数、节点颜色分类（论文、方法、任务等）、Tooltip 提示及漫游交互。 |
| `main_cut.js` | **核心逻辑文件**。负责加载数据、处理图谱数据结构、实现节点点击回调、搜索过滤逻辑及列表渲染。 |
| `data/standard_cut_2.json` | *(需自行准备)* 数据源文件，包含论文元数据及实体关系数据。 |
